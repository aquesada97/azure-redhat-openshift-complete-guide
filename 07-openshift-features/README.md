# 07 - OpenShift Features

## Module Overview

OpenShift provides a rich set of features beyond vanilla Kubernetes: a full web console, Source-to-Image builds, ImageStreams, the Operator Lifecycle Manager, and built-in templates. This module explores these OpenShift-specific capabilities.

---

## Learning Objectives

1. Navigate and use the OpenShift web console (both perspectives)
2. Build and deploy applications using Source-to-Image (S2I)
3. Use ImageStreams for image management and build triggers
4. Install and manage Operators via OLM
5. Use templates and Helm charts in OpenShift
