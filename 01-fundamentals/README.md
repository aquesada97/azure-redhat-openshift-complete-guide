# 01 - Fundamentals

## Module Overview

This module introduces the foundational concepts of Azure Red Hat OpenShift (ARO). Before diving into cluster creation and workloads, understand what ARO is, how it relates to Kubernetes and OpenShift, and how its architecture differs from vanilla Kubernetes distributions like AKS.

---

## Learning Objectives

By the end of this module you will:
1. Explain the relationship between Kubernetes, OpenShift Container Platform (OCP), and ARO
2. Identify key OpenShift capabilities absent in vanilla Kubernetes
3. Describe the ARO managed control plane architecture
4. List the core OpenShift components and their roles
5. Explain key differences between ARO, AKS, and self-managed OCP
6. Understand the Azure resource group structure created by ARO
