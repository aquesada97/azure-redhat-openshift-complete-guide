# 03 - Workloads

## Module Overview

This module covers deploying and managing all workload types in OpenShift — from simple stateless Deployments to StatefulSets, Jobs, and multi-container pod patterns.

---

## Learning Objectives

1. Deploy and manage Deployments with proper resource requests, probes, and update strategies
2. Configure Services for internal and external access
3. Create and manage OpenShift Routes with various TLS termination modes
4. Deploy StatefulSets, Jobs, CronJobs, and DaemonSets
5. Use init containers and sidecar patterns
