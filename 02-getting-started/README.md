# 02 - Getting Started

## Module Overview

This module walks through creating your first ARO cluster, connecting to it, and using the `oc` CLI effectively.

---

## Learning Objectives

1. Create an ARO cluster using `az aro create`
2. Connect to the cluster using `oc login`
3. Use `oc` CLI for common day-to-day operations
4. Access and navigate the OpenShift web console
5. Set up Azure AD as an identity provider
6. Manage kubeconfig contexts
