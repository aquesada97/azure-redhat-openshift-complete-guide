# 10 - Advanced Topics

## Module Overview

This module covers Day-2 operations and advanced scenarios: cluster upgrades, multi-tenancy patterns, disaster recovery with OADP/Velero, and GitOps with OpenShift Pipelines and ArgoCD.

---

## Learning Objectives

1. Plan and execute ARO cluster upgrades safely
2. Implement multi-tenancy with namespace isolation, quotas, and RBAC
3. Configure application backup and restore with OADP (Velero)
4. Implement GitOps with OpenShift GitOps (ArgoCD) and OpenShift Pipelines (Tekton)
